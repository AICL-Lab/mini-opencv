# Examples

Practical examples demonstrating Mini-OpenCV usage patterns.

## Available Examples

| Example | Description | Complexity |
|---------|-------------|------------|
| [Basic Usage](basic-usage) | Load, process, and save images | Beginner |
| [Pipeline Processing](pipeline-processing) | Batch processing with streams | Intermediate |

## Running Examples

After building the project:

```bash
# Basic example
./build/bin/basic_example

# Pipeline example
./build/bin/pipeline_example
```

## Example Source Code

All source code is available in the `examples/` directory:

- `examples/basic_example.cpp`
- `examples/pipeline_example.cpp`
